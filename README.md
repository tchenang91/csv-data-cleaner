# 🧹 CSV Data Cleaner
Kommandozeilen-Tool zur automatischen Bereinigung und Standardisierung von CSV-Dateien.

## Features
- Duplikaterkennung und -entfernung
- Fehlende Werte behandeln
- Datumsformat-Standardisierung
- Encoding-Korrektur
- Detaillierter Bereinigungsbericht

## Verwendung
```bash
python cleaner.py input.csv --output cleaned.csv --report
```
